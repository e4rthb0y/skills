# Gemini CLI Skills

A collection of specialized skills for the Gemini CLI agent. These skills are designed to automate common workflows, enforce standards, and enhance the agent's capabilities across different projects.

## Repository Structure

Each directory in this repository represents a standalone Gemini CLI skill:

- `[skill-name]/SKILL.md`: The primary definition and instructions for the skill.
- `[skill-name]/references/`: (Optional) Supporting documentation and detailed guidelines.

## Getting Started

To use these skills, ensure your environment is properly configured.

### Prerequisites

- **Gemini CLI**: The host agent for these skills.

## Contributing

When adding a new skill:

1. Create a new directory for the skill.
2. Include a `SKILL.md` file following the standard skill schema.
3. Add any necessary reference documentation in a `references/` subdirectory.
